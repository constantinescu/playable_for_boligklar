# 🎮 BoligKlar – Metaplayable Prototype

## 📌 Overview
This project is a **metaplayable interactive prototype** developed for the company **BoligKlar**.  
It is designed as an experimental web experience that demonstrates how interactive storytelling and lightweight gameplay mechanics can be used to engage users.

## 🎮 What is a Metaplayable?
A **metaplayable** is an interactive experience that combines:
- Lightweight gameplay elements
- Narrative or contextual interaction
- Product or concept exploration through play

This prototype explores how users can engage with a concept in a more interactive and playful way than traditional web pages.
